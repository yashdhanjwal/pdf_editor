# Free Online PDF Editor by Yash Dhanjwal

This is a modern, responsive, and SEO-optimized PDF Editor WebApp tool built with HTML, CSS, and JavaScript. All processing is done client-side for speed and security.

## Features

*   **Upload PDF:** Upload a PDF from your computer using a file picker or drag-and-drop.
*   **Dark/Light Mode:** Switch between dark and light themes for comfortable viewing.
*   **Page Navigation:** Easily navigate through the pages of your PDF.
*   **Add Text:** Add text to your PDF with customizable font, size, and color.
*   **Draw:** A freehand drawing tool with color and thickness options.
*   **Save & Download:** Download the edited PDF to your computer.

## Tech Stack

*   **HTML, CSS, JavaScript:** The core technologies for the web application.
*   **pdf-lib:** A JavaScript library for creating and modifying PDF documents.
*   **pdf.js:** A web standards-based platform for parsing and rendering PDFs.
*   **interact.js:** A JavaScript library for drag and drop, resizing, and multi-touch gestures.
